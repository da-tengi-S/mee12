## 🎨 Customization Quick Reference

### Change Colors
Edit in `styles.css` (Line 10-17):
```css
--color-primary: #D4AF37;     /* Gold */
--color-background: #0d0d0d;  /* Black */
--color-accent: #ff6b6b;      /* Red */
```

### Change Fonts
Current fonts:
- **Headings**: Playfair Display (elegant serif)
- **Body**: Space Mono (modern mono)

To change: Edit Google Fonts link in `index.html` (Line 12-13)

### Update Statistics
In `index.html`, find data-target values:
```html
<span class="stat-number" data-target="50">0</span>
```
Change `50` to your number.

### Update Skills
Find skill-progress data-progress values:
```html
<div class="skill-progress" data-progress="95"></div>
```
Change `95` to your percentage (0-100).

## 🐛 Common Issues & Fixes

**Images not showing?**
- Check images folder exists
- Verify file names match (case-sensitive)
- Check file extensions (.jpg not .jpeg)

**Mobile menu not working?**
- Clear browser cache
- Check script.js is loaded
- Try different browser

**Site looks broken?**
- Clear cache (Ctrl+Shift+Delete)
- Check all 3 files in same folder
- View browser console for errors

**Changes not appearing?**
- Hard refresh (Ctrl+Shift+R)
- Clear browser cache
- Wait 1-2 minutes if deployed
